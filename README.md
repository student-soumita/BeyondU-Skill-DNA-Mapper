# 🚀 BeyondU – Skill DNA Mapper

BeyondU – Skill DNA Mapper is a full-stack career development platform that helps users analyze, track, and improve their technical skills through an interactive dashboard. It combines skill assessment, career planning, project evaluation, and personalized recommendations into one unified experience.

---

## ✨ Features

- 🔐 User Authentication (Login, Registration & Password Reset)
- 📊 Interactive Skill DNA Dashboard
- 📈 Skill Level Management
- 🧬 Skill DNA Mapping
- 💼 Dream Company Match
- 🎯 Hidden Talent Discovery
- 🤖 Intelligent Chatbot for Learning & Guidance
- 📂 Project Reviewer
- 📝 Skill Verification System
- 🎮 Daily Skill Challenges
- 👤 Profile Management
- 📄 Downloadable Progress Report
- 🎬 Cinematic Intro Experience
- 🚪 Secure Logout System

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/student-soumita/BeyondU-Skill-DNA-Mapper.git
```

Move into the project folder:

```bash
cd BeyondU-Skill-DNA-Mapper
```

Install dependencies:

```bash
pip install flask scikit-learn pandas numpy joblib
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
